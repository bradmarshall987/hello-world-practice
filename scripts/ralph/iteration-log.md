# Ralph Iteration Log

This file records each Ralph iteration for debugging and context. Each Amp instance appends an entry after completing (or attempting) a story.

## Log Format

Each entry should follow this format:

```markdown
## Iteration N - YYYY-MM-DD HH:MM:SS

**Story:** US-XXX - Story Title
**Status:** ✅ Complete | ⚠️ Partial | ❌ Failed
**Summary:** Brief description of what was done

**Files Modified:**
- path/to/file1.ts
- path/to/file2.ts

**Notes:** Any relevant context or issues encountered
```
